## 🐛 Troubleshooting

### GitHub Pages not showing up
- Wait 2-3 minutes after enabling
- Hard refresh: Ctrl+Shift+R
- Check Settings → Pages again

### Build failed in Actions
- Check Actions tab for error message
- Most likely: typo in build script
- Solution: Fix and push again

### Offline file won't open
- Ensure it's HTML file (not zip)
- Try different browser
- Check file size (~5.9 MB)

### Features not working
- Check browser console (F12)
- Look for JavaScript errors
- Ensure JavaScript is enabled
